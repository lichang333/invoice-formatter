# 发票排版工具

一个本地运行的电子发票 PDF 双拼排版工具。上传多张发票 PDF，或粘贴可公开读取的发票 PDF 链接后，工具会把每两张发票排到一张 A4 页面中，生成可下载的排版 PDF，适合批量打印和节省纸张。

## 功能

- 支持拖拽或点击批量上传 PDF 发票
- 支持粘贴 PDF 发票链接导入
- 自动按每 2 张发票生成 1 页 A4 输出
- 支持纵向上下排版和横向左右排版
- 可调整页边距、发票间距和缩放比例
- 生成后的 PDF 在浏览器中直接下载

## 使用方式

直接用浏览器打开 `index.html`：

```bash
open index.html
```

也可以把文件部署到任意静态网页托管服务。

## 链接导入说明

在线版本可以读取允许浏览器跨域访问的 PDF 链接。若发票平台禁止跨域读取、链接需要登录态、或链接不是直接 PDF 文件，浏览器会拦截下载；这种情况下请先下载 PDF，再拖拽上传。

## 技术说明

当前 v1 是纯前端实现，不需要服务端。PDF 处理在浏览器本地完成，使用 `pdf-lib` 读取并重排发票页面。
